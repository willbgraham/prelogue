# Cast — Demo Runbook

**The one-line pitch:** *"Upload a screenplay, hear it performed instantly with AI voices, and let real actors audition right into it — a living table read."*

---

## Before the demo (one-time setup)

- [ ] **ElevenLabs credits topped up**, then pre-warm the demo script to 100% (ping me to re-run — it resumes from ~50%).
- [ ] **Mark a Writer's Choice** on the demo script: Casting Dashboard → pick the take → **Writer's Choice**. *(Without this the table read is all-AI and your recorded actor clip won't appear.)*
- [ ] **Fix the Yonaburu split line** in the Line Editor (merge *"insurance. Mrs. Smith."* back into Yonaburu's line).
- [ ] App running on your **device** (Expo Go / dev build), signed in as the **writer (Pro)**. Good Wi-Fi + speaker or headphones.

---

## The 4-minute flow

1. **Discover → open "All You Need Is Kill."** "A real screenplay, auto-parsed into scenes, characters, and lines."
2. **Tap "Play with AI Voices" → "Play Table Read."** Let it run a beat: each character in a distinct AI voice, and the **narration types onto a screenplay page** as it's read. *(Pre-warmed = instant.)* — **this is the hook.**
3. **Show the hybrid.** When it reaches the cast role, your **recorded actor video plays inline**. "AI fills every part; real actors slot into theirs."
4. **Switch to Actor** (role switcher) → open a role → **Read for this role** → tap **AI Voices** (scene partner) → **Record** a line → **Done** → **Review** → **Submit**. "Actors audition gap-free, line by line — no dead air."
5. **Back as the writer** → the take shows in **Casting** → mark **Writer's Choice** → it's now in the table read. Close the loop.

---

## If something glitches
- **A line is silent** → not yet voiced; it auto-advances. (Top up credits + re-warm to remove gaps.)
- **Recording** → start performing *after* the red dot appears; tap **Done** to advance.
- **Don't navigate away mid-generation** — let the progress bar finish (leaving pauses it).

## Don't show (not demo-ready)
- Upgrade / Stripe (off by design), Line Editor internals, Settings, the second test script.
